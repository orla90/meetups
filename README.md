# Meetups
Responsive Angular application for organizing meetups. Implemented access for two roles: user and admin. Users and admins can subscribe / unsubscribe from other user's meetups, create, update and delete meetups. Admins can also edit user list (add, update, delete)

### Deploy
[Meetups](https://orla90.github.io/meetups)

### Stack
- [Angular](https://angular.io/)
- [Angular Material](https://material.angular.io/)
